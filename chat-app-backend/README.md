## Description

This is our chat application demo with the backend source code

## Technologies

Node v17.5.0
Postgresql
NestJS
Prisma
GraphQL

## Installation

```bash
cp .env.example .env
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
